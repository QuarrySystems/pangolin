# @quarry-systems/agora-providers-fargate

Scaffold package — future home of the AWS Fargate `ComputeProvider` implementation.

The Fargate ComputeProvider will be implemented in DAG 2 and will expose an `@aws-sdk/client-ecs`-backed compute provider conforming to the `ComputeProvider` interface defined in `@quarry-systems/agora-core`.

## Status

Scaffold only. No public API yet.

## Dependencies

- `@quarry-systems/agora-core` — workspace peer providing core types and interfaces.
