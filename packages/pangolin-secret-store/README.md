# -systems/pangolin-secret-store

`SecretStore` seam plus implementations — `InlineSecretStager` (AWS Secrets Manager) and `LocalSecretStore` (on-disk staging) for per-dispatch inline secrets.

```bash
pnpm add -systems/pangolin-secret-store
```
